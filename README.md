# Vaciados y Mudanzas Miranda

Sitio web corporativo para **Vaciados y Mudanzas Miranda**, empresa especializada en mudanzas, vaciados, portes y limpieza integral en **Miranda de Ebro**.  

El sitio está diseñado para mostrar los servicios de la empresa de manera profesional, con optimización SEO local y una experiencia de usuario moderna.

---

## Características

- **Secciones principales**: Inicio, Conócenos, Limpieza, Vaciados, Portes, Mudanzas, FAQ.
- **Formulario de presupuesto**: formulario profesional con validación y campo de teléfono opcional.
- **Botones flotantes**: contacto rápido por WhatsApp y acceso directo al formulario de presupuesto.
- **Hero con overlay**: imagen de portada con logo centrado y eslogan.
- **Menús de navegación**: intuitivos con recuadros y efecto hover.
- **Sección FAQ interactiva**: acordeones para preguntas frecuentes.
- **Optimización SEO**: metadatos, keywords, descripciones y datos estructurados JSON-LD para negocio local.
- **Diseño responsivo**: adaptado a dispositivos móviles, tablets y pantallas grandes.
- **Estilo visual profesional**: recuadros blancos con sombras, bordes redondeados, hover effects y tipografía moderna.

---

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox y Grid)
- JavaScript (vanilla)
- Google Fonts (Montserrat)
- JSON-LD para SEO local
